# Log Data Analysis

A simple project to extract, transform, and analyze system log data using Python, SQL, and Linux.

## Project Overview
This project automates the process of reading system logs, cleaning the data, loading it into a database, and generating basic analytical reports.

## Features
- Extracts log data from Linux system files  
- Cleans and structures raw logs  
- Stores data in a SQL database  
- Generates summary reports and insights  

## Technologies Used
- **Python**
- **MySQL / SQLite**
- **Linux**
- **Shell Scripting**
